# State Machines

## V1 states
`DISCOVERED`, `EVALUATED`, `APPLY_READY`, `APPLIED_UNSIGNED`, `APPLIED_CONFIRMED`, `EXECUTION_READY`, `EXECUTED`, `VALIDATED`, `PUBLISHED`, `SUBMISSION_READY`, `SUBMITTED_UNSIGNED`, `RECONCILING`, `DONE`, `FAILED`, `BLOCKED_RETRIEVAL_REQUIRED`, `BLOCKED_OPERATOR_REVIEW`, `BLOCKED_ARTIFACT_EMISSION_REQUIRED`.

## V1 legal transition highlights
- `DISCOVERED -> EVALUATED`
- `EVALUATED -> APPLY_READY | BLOCKED_RETRIEVAL_REQUIRED`
- `APPLY_READY -> APPLIED_UNSIGNED | BLOCKED_OPERATOR_REVIEW`
- `APPLIED_UNSIGNED -> APPLIED_CONFIRMED | BLOCKED_OPERATOR_REVIEW`
- `APPLIED_CONFIRMED -> EXECUTION_READY`
- `EXECUTION_READY -> EXECUTED | BLOCKED_RETRIEVAL_REQUIRED`
- `EXECUTED -> VALIDATED -> PUBLISHED -> SUBMISSION_READY -> SUBMITTED_UNSIGNED -> RECONCILING -> DONE`
- Any non-terminal state may transition to `FAILED` when a hard failure occurs.

## Prime states
`DISCOVERED`, `EVALUATED`, `COMMIT_READY`, `COMMITTED_UNSIGNED`, `COMMIT_CONFIRMED`, `REVEAL_READY`, `REVEALED_UNSIGNED`, `REVEAL_CONFIRMED`, `SHORTLIST_PENDING`, `FINALIST`, `FINALIST_ACCEPT_READY`, `FINALIST_ACCEPTED_UNSIGNED`, `FINALIST_ACCEPT_CONFIRMED`, `TRIAL_EXECUTION_READY`, `TRIAL_EXECUTED`, `TRIAL_PUBLISHED`, `VALIDATOR_SCORING_PENDING`, `WINNER_PENDING`, `WINNER_DESIGNATED`, `FALLBACK_PROMOTABLE`, `DONE`, `FAILED`, `BLOCKED_DEADLINE`, `BLOCKED_OPERATOR_REVIEW`, `BLOCKED_RETRIEVAL_REQUIRED`, `BLOCKED_ARTIFACT_EMISSION_REQUIRED`.

## Prime legal transition highlights
- Discovery/evaluation: `DISCOVERED -> EVALUATED -> COMMIT_READY`
- Commit window: `COMMIT_READY -> COMMITTED_UNSIGNED -> COMMIT_CONFIRMED`
- Reveal window: `COMMIT_CONFIRMED -> REVEAL_READY -> REVEALED_UNSIGNED -> REVEAL_CONFIRMED`
- Shortlist/finalist: `REVEAL_CONFIRMED -> SHORTLIST_PENDING -> FINALIST -> FINALIST_ACCEPT_READY`
- Finalist acceptance: `FINALIST_ACCEPT_READY -> FINALIST_ACCEPTED_UNSIGNED -> FINALIST_ACCEPT_CONFIRMED`
- Trial: `FINALIST_ACCEPT_CONFIRMED -> TRIAL_EXECUTION_READY -> TRIAL_EXECUTED -> TRIAL_PUBLISHED`
- Scoring/winner: `TRIAL_PUBLISHED -> VALIDATOR_SCORING_PENDING -> WINNER_PENDING -> WINNER_DESIGNATED -> DONE`
- Fallback: `SHORTLIST_PENDING | WINNER_PENDING -> FALLBACK_PROMOTABLE -> FINALIST | DONE`

## Blocked states and causes
- `BLOCKED_RETRIEVAL_REQUIRED`: retrieval packet missing for retrieval-mandatory families.
- `BLOCKED_OPERATOR_REVIEW`: operator approval missing at irreversible unsigned handoff edges.
- `BLOCKED_ARTIFACT_EMISSION_REQUIRED`: canonical bundle incomplete before review/completion edges.
- `BLOCKED_DEADLINE` (Prime): any phase guard failed (`commit`, `reveal`, `finalist accept`, `trial`, `score commit`, `score reveal`, `fallback promotion`).
